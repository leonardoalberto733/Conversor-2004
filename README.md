# Conversor-2004
Use *npx live-server* para rodar

Para Kanji:

    - digite apenas a palvra

    - Ex.: porta

Para Hangul:
    - coloque um asterisco no começo da palavra

    - separe as partes da palvra que vão virar cada uma uma letra por ponto 

    - Ex.: *tra.ns.for.ma.ção

Para Katakana
    - coloque uma cerquilha no começo da palavra

    - separe as partes da palvra que vão virar cada uma uma letra por ponto 

    - Ex.: #ma.ri.a

Para texto literal
    - coloque uma barra invertina no começo da palavra

    - Ex.: \RPG

Enter para converter;

Ctrl + Backspace para apagar o texto já convertido;
